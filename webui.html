Output Format: VGA / YPbPr (Component Video)

The default output mode is VGA, suitable for VGA cables or HDMI converters.

A new YPbPr mode can also be selected, but compatibility with some options is still spotty.

Load on Start: Default Presets

These are the default presets that gbs-control ships with. Pick one that works well with your source and TV.

Load on Start: Custom Presets

These slots hold your saved presets. Select the slot you want to save to / to be loaded by default.


Save current Settings as Custom Preset

This allows you to store your image tweaks in a new preset file. These Custom Presets can be automatically loaded on startup.

There are 5 slots available. PAL and NTSC (50Hz, 60Hz) sources are assigned automatically.

Active FrameTime Lock (not compatible with all displays)

This option keeps the input and output timings aligned, fixing the horizontal tear line that can appear sometimes.

Two methods are available. Try switching methods if your display goes blank.
